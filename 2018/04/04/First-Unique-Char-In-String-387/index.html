LeetCode#387. First Unique Character in a String

另一种解法

这种解法是通过找到每个字母第一次出现的位置和最后一次出现的位置，并将这两个位置进行对比，如果一样就说明只出现了一次。在LeetCode的这道题中，这种解法的速度会更快。不过在极端情况下，这种解法速度会降低到26 * s.length()，而第一种解法永远只需要2 * s.length()次。
